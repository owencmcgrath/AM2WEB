Where do I find my Apple Music XML file?
Open Apple Music, go to File → Library → Export Library. Save the file to your computer, and then upload it here.
Is Spotify/Soundcloud/Pandora/etc. supported
This only explicitly supports XML files exported from Apple Music, but Spotify and other service might work, it all depends on the XML file.
Is there a file size limit?
Yes, the maximum file size is 50MB.
Is Cider supported?
Cider, as far as I know, does not allow for exporting the library as an XML file. If you know how to do this, please open a GitHub issue.
Is any personal information stored?
No, the only information stored is the data from the XML file you upload with the songs from your Apple Music library.
How was this built?
Flask, HTML/CSS, and a MySQL backend. Claude helped too, allegedly ;).
